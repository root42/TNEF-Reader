TNEF Reader
(C)2012 Arne Schmitz <arne.schmitz@gmx.net>

This is an iOS application to decode TNEF encoded attachments (aka winmail.dat), which 
is used for example by Microsoft Outlook.

After installing the app, you can open winmail.dat files from within Mail.app and viewing
their contents.

TNEF Reader utilizes the tnef program by Mark Simpson et al:

http://tnef.sourceforge.net/
